<h1>Project Info</h1>
Project name</strong>: my-library.
</br>
Description</strong>: Personal library webp app to track owned books, their format, if the user have read them or not yet, and finally search for book cover in openlib and display it.
</br>

<h1>Tech Stack</h1>
1. DB: postgres
2. Backend: Express js
3. Front end: HTML CSS Javascrip Embedded Js

<h1>How to</h1>
To initiate the project</strong>:
</br>
1.download or clone the project files.
</br>
2. in project root run the code
</br>
using npm:
<code>npm install</code>
using pnpm:
<code>pnpm install</code>
</br>
3. use the booklistdbbackup to create DB in postgres.
</br>
4.connect the db using an env file.
</br>
5.in project root run the code
</br>
<code>node index.js</code>
</br>
